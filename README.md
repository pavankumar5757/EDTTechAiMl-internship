## 📌 Objective

The objective of this task is to explore and understand a dataset using Python. This includes identifying data types, checking for missing values, and analyzing the overall structure of the data. This step is part of Exploratory Data Analysis (EDA) and is essential before building machine learning models.

 Tools & Technologies

* Python
* Pandas
* NumPy
* Vs code

 Dataset

* File Name: `data.csv`
* Format: CSV
* Description: The dataset contains structured data with both numerical and categorical features.

 Steps Performed

1. Loaded the dataset using Pandas
2. Displayed the first few rows using `head()`
3. Analyzed dataset structure using `info()`
4. Generated statistical summary using `describe()`
5. Identified data types of each column
6. Checked for missing values
7. Separated numerical and categorical columns
8. Extracted unique values from categorical columns
9. Determined dataset shape (rows and columns)

 Key Observations

* The dataset contains both numerical and categorical data
* Data types include integer, float, and object
* Missing values were identified in some columns (if any)
* The dataset structure was successfully analyzed

 Deliverables

 Final Outcome

This task helped in understanding the dataset structure and data types. It builds a strong foundation for further steps like data cleaning, visualization, and machine learning model development.

Conclusion

Understanding the dataset is the first and most important step in any data science project. Proper analysis ensures better preprocessing and improves the performance of machine learning models.
